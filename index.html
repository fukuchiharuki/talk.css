What is this?

talk.css makes paragraphs look like a talk app.

Wow! How to use it?

It is easy to use. You only need to use 3 classes: .talkcss, .right, .left.

Sounds great.

How to use

Link talk.css to your HTML: 

<link href="/path/to/talk.css" rel="stylesheet" />

And add the class talkcss to an element to make a talk area: 

<div class="talkcss">
</div>

And append paragraphs into the area: 

<div class="talkcss">
	<p class="right">This will be set on right.
	<p class="left">This will be set on left.
</div>
